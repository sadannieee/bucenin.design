```
bucenin/
├── index.html              — Главная страница
├── css/
│   └── style.css           — Глобальные стили
├── assets/
│   ├── logo-color.png      — Цветной логотип (основной)
│   ├── logo-white.png      — Белый логотип (для тёмных фонов)
│   ├── logo-dark.png       — Тёмный логотип
│   ├── logo-color-full.png — Цветной логотип полный вариант
│   ├── logo-black.png      — Чёрный логотип
│   └── logo-black-full.png — Чёрный логотип полный вариант
└── pages/
    ├── portfolio.html      — Портфолио
    ├── services.html       — Услуги и прайс
    ├── reviews.html        — Отзывы
    ├── about.html          — Об авторе
    └── contact.html        — Контакты и форма заявки
```

---

## Откуда берутся отзывы

**Источник:** все отзывы захардкожены прямо в HTML.

Файлы: `index.html` (секция `#reviews`, 3 карточки) и `pages/reviews.html` (6 карточек).

Каждая карточка содержит:
- Имя клиента и инициалы (аватар)
- Текст отзыва
- Категория услуги (тег)
- Пометка «ВКонтакте · обсуждения · 2024»

Отзывы взяты из обсуждений VK-сообщества студии.
Чтобы добавить новый отзыв — скопировать блок `.rev-card` в `pages/reviews.html` и отредактировать текст.

## Откуда берутся услуги

**Источник:** захардкожены в HTML.

Файлы: `index.html` (секция `#services`, 4 карточки) и `pages/services.html` (детальное описание каждой услуги).

Список услуг:
1. **Логотип и брендбук** — 3 500–5 500 ₽
2. **Дизайн сообщества / UI kit** — персональный заказ
3. **Инфографика и маркетплейсы** — персональный заказ
4. **GTA / SAMP дополнения** — 4 999 ₽ (фиксированная цена)

Чтобы изменить цену или описание — редактировать HTML напрямую в `pages/services.html`.

## Откуда берётся портфолио

**Источник:** захардкожено в HTML.

Файлы: `index.html` (секция `#portfolio`, 4 превью-кейса) и `pages/portfolio.html` (6 полных кейсов с фильтрами).

Кейсы:
1. МастерКом — Upcycling (брендинг)
2. Федерация фиджитал спорта Рязани (брендинг)
3. BET — Telegram бот (UI/UX)
4. YouTube-канал «Амазяев» (соцсети)
5. SAMP — UI для сервера (GTA/SAMP)
6. Карточки и инфографика для маркетплейсов (инфографика)

Фильтрация по категориям работает через JS: кнопки `.filter-btn` с атрибутом `data-filter` управляют видимостью карточек `.port-item[data-cat]`.
